<p>
  Alurabooks é um projeto fictício criado durante o curso de HTML e CSS da Alura, focado em responsividade e mobile-first.

Durante a construção entendi como acessar, visualizar e interpretar um protótipo de site criado no Figma. 

Também tive meu primeiro contato com Flexbox, associei arquivos CSS através do @import, desenvolvi um menu hambúrguer interativo sem a necessidade de JavaScript, importei fontes do google fonts, editei input e seu placeholder.

Tudo isso ocorreu a partir da abordagem mobile-first e logo aprendi a utilizar media queries na construção de layouts que respondessem também as telas de tablets e desktops.

Foi uma grande jornada e, por fim, o site foi publicado através da plataforma Vercel.
</p>
